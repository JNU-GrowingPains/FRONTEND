## 🚀 빠른 시작

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 🔐 기본 로그인 정보 (개발 환경)

**통일된 계정 정보:**
```
이메일: kim@suelo.co.kr
비밀번호: 아무 비밀번호 (Mock 모드)
이름: 김슈엘
쇼핑몰명: 슈엘로 화장품
```

이 정보는 다음 세 곳에서 모두 동일하게 사용됩니다:
1. **로그인 페이지** - 기본 계정
2. **상단 네비게이션** - 사용자 표시
3. **계정 관리 페이지** - 프로필 정보

> ⚠️ **주의**: 실제 백엔드 API 연동 전까지는 Mock 인증이 사용됩니다.
> 자세한 내용은 [API 연동 가이드](./04-api-integration.md)를 참고하세요.

## 📂 주요 페이지

- **로그인 페이지**: 웹앱 진입 시 최초 화면
- **회원가입 페이지**: 신규 사용자 등록
- **비밀번호 찾기 페이지**: 비밀번호 재설정 링크 전송
- **재구매 분석**: 재구매율, 고객별/상품별 재구매 데이터 분석 (기본 페이지)
- **상품 분석**: 상품별 판매 데이터와 리뷰 분석
- **고객 분석**: 고객 등급별 분포 및 구매 패턴 분석
- **계정 관리**: 관리자 정보 및 설정

## 🔄 데이터 흐름

### Mock 모드 (현재)
```
로그인 → Zustand Store → localStorage
         ↓
    상단/계정관리에 동일 정보 표시
```

### Production 모드 (백엔드 연동 후)
```
로그인 → 실제 API → JWT 토큰
         ↓
    Zustand Store → localStorage
         ↓
    상단/계정관리에 실제 사용자 정보 표시
```
